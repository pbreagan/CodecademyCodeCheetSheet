Coding Cheet Sheet

General HTML

Tag Name Description Display
<h1> - <h6> Headers Header elements sizes vary. Largest is header1 and smallest is header6. Block
<p> Paragraph The paragraph element holds text and can be styled via CSS. Block
<div> div (division) The div element does not any specific use, but can be used as "containers" other HTML elements. Block
<body> Body The body element contains all the visible HTML elements read by the browser. Block
<head> Head The head element contains all the non visible HTML elements read by the browser. This includes links to stylesheets and other meta data. NA
<link> Link The link element is used to link resources to the html page, such as CSS style sheets, JavaScript sheets, and font styles. NA
<a> Anchor The anchor element is used wrapped around other elements to link them to other webpages. Inline
<header> Header The header element is used to store HTML elements used at the top of the webpage. Usually navigation links and/or heading elements. Block
<footer> Footer The footer element is used to store HTML elements used at the bottom of the webpage. Usually elements that hold text such as contact information. Block

Tables

Tag Name Description
<table> Table The element to create and hold all other table elements.
<thead> Table Head The element gives the header to a group of a table.
<tbody> Table Body Holds the main contents of the table.
<tr> Table Row Holds any number of table cells
<td> Table Data Holds data in a singular cell.
<th> Table Header Gives the header to a specific cell.

CSS

Property Description
color: Changes the foreground color of the targeted element.
background-color: Changes the background color of the targeted element.
font-family: Changes the font for the targeted element.
height: Changes the height of the targeted element.
width: Changes the width of the targeted element.
padding: Changes the distance between the content and the border of the targeted element.
border: Changes the border properties of the targeted element. eg; 1px solid black
margin: Changes the distance between the border of the targeted element to another element. top, right, bottom, left.
